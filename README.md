# Marketing Campaign Performance Dashboard – Power BI Project

A fully interactive **Power BI dashboard** designed to analyze and compare the performance of multiple marketing campaigns across platforms such as **Google Ads, Facebook, Instagram, LinkedIn, and YouTube**.
This project demonstrates expertise in **data modeling, DAX measures, KPI tracking, marketing funnel analysis, and ROI evaluation** using a hypothetical dataset created specifically for this analysis.

---

## Project Overview

This project focuses on measuring the effectiveness of marketing campaigns by analyzing key performance metrics such as:

* **Total Cost & Total Revenue**
* **ROI% (Return on Investment)**
* **CPL (Cost Per Lead)**
* **Conversion Rate**
* **Marketing Funnel Performance**
* **Platform-wise Campaign Efficiency**
* **Audience Insights (Age Group Analysis)**
* **Monthly Revenue Trends**

The dashboard provides a comprehensive view for marketers and decision-makers to optimize spending and improve campaign ROI.

---

## Key Dashboards Included

### **1. Marketing Campaign Performance Overview**

## Dataset Information

This project uses a **hypothetical dataset created using ChatGPT**.
It includes multiple tables structured specifically for marketing analytics and Power BI modeling:

### **Tables Used**

* **CampaignMeta** – Campaign names, managers, target audience groups
* **CampaignPerformance** – Cost, revenue, impressions, clicks, leads, applications, enrollments
* **ChannelRates** – CPC, CPM, platform-specific cost values
* **Dates** – Standard date table used for time intelligence
* **Measures_Table** – DAX measures including:

  * Total Impressions
  * Total Clicks
  * Total Leads
  * Total Cost
  * Total Revenue
  * ROI%
  * CPL
  * Conversion Rate
  * Average CPC
  * Average CPM

**Note:**
This dataset is *artificial* and generated only for learning, demonstration, and portfolio purposes.
